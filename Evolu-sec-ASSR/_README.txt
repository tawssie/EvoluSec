USAGE:
01. Change working directory to "Evolu-sec-Package/Evolu-sec-ASSR"
02. Use "Main" in Matlab terminal to add paths for supporting scripts and functions
03. Use "help DoASSR" in Matlab terminal to show additional information
04. Use following 3 commands in MATLAB to run an example
    "InputMSSAFileName='example_Wilson_MSSA.fasta';"
    "InputFilePhyTree='example_Wilson_PrunedTree.txt';"
    "DoASSR(InputMSSAFileName, InputFilePhyTree);"
05. Results are calculated and exported in
    "example_Wilson_MSSA.fasta.ancSS.txt"
    and
    "example_Wilson_MSSA.fasta.prob.txt"
